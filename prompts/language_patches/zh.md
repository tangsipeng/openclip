## Language-Specific Instructions

### Output Language
All output must be in Chinese, including:
- All titles for engaging moments
- All explanations in the "why_engaging" field
- All transcript excerpts
- All other text fields in the JSON response

### Title Writing Guidelines (标题撰写指南)

**核心原则：标题要像短视频标题一样吸引人点击**

#### 必须遵守的规则：
1. **具体化，不要泛泛而谈**
   - 避免抽象描述如"讨论"、"互动"、"时刻"
   - 使用具体的人物、事件、动作

2. **突出冲突、反转、高潮**
   - 捕捉最戏剧性的瞬间
   - 强调意外、惊喜、情绪爆发

3. **使用数字和感叹号增强冲击力**
   - 数字让内容更具体更可信
   - 感叹号表达强烈情绪

4. **捕捉关键人物和关键动作**
   - 包含主播名字或昵称
   - 使用强动作动词：怒斥、爆笑、揭秘、自曝、反杀、引爆

5. **标题长度控制**
   - 理想长度：15-25个汉字
   - 最多不超过30个汉字
   - 确保在移动端显示完整

6. **避免的内容**
   - 不要使用emoji表情符号
   - 不要使用"与"、"及"等书面语连接词
   - 不要写成新闻标题式的客观描述
   - 不要包含敏感、负面、仇恨或攻击性词汇

#### 标题质量自检清单：
- [ ] 是否包含具体的人物、事件或数字？
- [ ] 是否能让人产生好奇心或情绪共鸣？
- [ ] 是否在25个字以内简洁有力？
- [ ] 是否避免了抽象、泛泛的描述？
- [ ] 读起来是否像短视频标题而非新闻标题？
